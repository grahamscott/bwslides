Dropping 3.9

What is 3.9?

  1. Legacy app that predates the current web app
  2. When we released the current app we left it running so people could still access it
  3. It's unmaintained and unsupported

What's changing?

  1. Until recently we linked to it from the top-right navigation
  2. We're now removing direct access via the /onebrand url
  3. We'll no longer bounce users on old browsers to 3.9

Users have been informed of these changes

Legacy browsers

We let them know that they need to upgrade to use the app
